🏢 Smart HR Leave & Attendance Management System
📌 Overview

The Smart HR Leave & Attendance Management System is a business application built using Zoho Creator (Zia AI App Builder) to automate employee leave processing and attendance tracking.

The application streamlines HR operations by implementing structured data management, automated workflows, validation rules, and role-based access control.

This project demonstrates low-code development combined with business logic automation.

🚀 Features
👥 Employee Management

Store employee details (ID, department, role, joining date)

Maintain annual and sick leave balances

Centralized employee database

📝 Leave Management

Employees can submit leave requests

Automatic calculation of number of leave days

Leave balance validation before submission

Manager approval/rejection workflow

Automatic deduction of leave balance after approval

Email notifications for status updates

🕒 Attendance Tracking

Daily check-in and check-out logging

Automatic working hours calculation

Late arrival detection (after 9:30 AM)

Flagging insufficient working hours (< 8 hours)

📊 Dashboard & Reports

Total employee count

Employees currently on leave

Late arrival statistics

Department-wise attendance report

Leave balance summary

🔐 Role-Based Access Control

Admin – Full access to all modules

Manager – Approve/reject leave requests

Employee – Apply leave and view personal attendance

⚙️ Technologies Used

Zoho Creator

Zia AI App Builder

Deluge Scripting (Workflow Automation)

Role-Based Access Control

Dashboard & Reporting Tools

🧠 Automation Logic

Prevent leave submission if balance is insufficient

Deduct leave balance automatically after approval

Mark employee as "Late" if check-in time is after 9:30 AM

Flag attendance record if working hours are less than 8 hours

Send automated email notifications for leave request actions

Automate HR processes
